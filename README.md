# 🌐 Personal Portfolio Website

A responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript** to showcase my skills, projects, education, and contact information.

🔗 https://lipsawebsite.vercel.app/

## 📖 About the Project

This portfolio website serves as my personal online presence where visitors can:

- Learn about me
- View my technical skills
- Explore my projects
- Download my resume
- Contact me through the contact section

The website is fully responsive and works across desktop, tablet, and mobile devices.

## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/Lipsapadhy/mywebsite.git
```

Go to the project folder:

```bash
cd mywebsite
```

Open `index.html` in your browser.
